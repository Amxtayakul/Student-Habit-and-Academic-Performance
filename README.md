# Student-Habit-and-Academic-Performance
R program that demonstrates the Univariate and Bivariate Analysis for Student Habits and Academic Performance

# Prerequisites:
Download the CSV file: https://www.kaggle.com/datasets/aryan208/student-habits-and-academic-performance-dataset
 - Optional to convert the CSV file into an Excel (xlxs) file.

# Imported R Libraries:
 - readxl - to import the Excel file used for univariate and bivariate computation and analysis
 - dplyr - for data manipulation and tabulation
 - ggplot2 - used for visual presentation of the computed data
